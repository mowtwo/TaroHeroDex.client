# 基于 Taro 的风暴英雄图鉴小程序

这是一个练手项目，意在本人练习使用 Taro 以及复习 React

## 技术选择

- Taro3
- React17+
- Mobx

## 快速开始

### 下载

```bash
git clone https://github.com/mowtwo/TaroHeroDex.client
```

### 依赖安装

```bash
npm i
# or yarn
yarn
```

### 运行

此项目暂时只兼容微信小程序

```bash
npm run dev:weapp
# or yarn
yarn dev:weapp
```
